Instrucciones de GitHub

Cómo crear una cuenta y repositorio en GitHub

1. Crear una cuenta en GitHub

Para comenzar a usar GitHub, primero necesitas crear una cuenta:

  1. Ve a github.com
  2. Haz clic en "Sign up" en la esquina superior derecha
  3. Ingresa tu correo electrónico, crea una contraseña segura y elige un nombre de usuario
  4. Verifica tu dirección de correo electrónico
  5. ¡Listo! Ya tienes una cuenta de GitHub
